# MTShop-doc
仿锤子商城-文档



# 技术栈

PC前端：vue全家桶 vue+vue-router-vuex，http请求使用axios

移动前端：同PC

小程序：小程序自身技术栈

后台：同PC，UI框架会用到iview

后端接口：ThinkPHP V5.0 实现Rest API
